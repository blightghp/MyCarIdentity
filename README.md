# MyCarIdentity 🏎️

**Qual carro reflete a sua personalidade? Descubra!**

Um app web que cruza seus dados financeiros, estilo de vida e preferências pessoais com um banco de dados de carros do mercado brasileiro e te recomenda os carros que mais combinam com você — de verdade.

## Perguntas que o app responde

- 🤔 **Qual carro reflete a minha identidade e por quê?**
- 💰 **Com base na minha renda e capacidade de investimento, qual carro combina comigo?**

## Stack

| Frontend | Backend | Banco |
|----------|---------|-------|
| React + TypeScript + Vite | Python + FastAPI | SQLite |

## Como rodar

```bash
# Backend
cd backend
.\.venv\Scripts\activate   # Windows
uvicorn main:app --reload

# Frontend (em outro terminal)
cd frontend
npm run dev
```
